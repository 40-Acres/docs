## Base URL

All endpoints are served from:

```text
https://40acres.finance/api
```

## Supported Protocols

All endpoints that accept a `protocolName` parameter support the following values:

| Protocol | Chain | Value |
| --- | --- | --- |
| Aerodrome | Base | `aerodrome` |
| Velodrome | Optimism | `velodrome` |
| Pharaoh | Avalanche | `pharaoh` |
| Blackhole | Avalanche | `blackhole` |
