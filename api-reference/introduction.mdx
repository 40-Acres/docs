## Base URL

All endpoints are served from: https://app.40acres.finance/api

## Supported Protocols

| Protocol | Chain | Value |
|----------|-------|-------|
| Aerodrome | Base | `aerodrome` |
| Velodrome | Optimism | `velodrome` |
| Pharaoh | Avalanche | `pharaoh` |
| Blackhole | Avalanche | `blackhole` |
